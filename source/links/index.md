### 如何交换友链

请按以下格式发送邮件至主页邮箱

```yaml
- name: 名字
  link: 博客链接
  description: 描述
  avatar: 头像链接
```

### 额外要求
- 先添加本站友链
- 支持HTTPS
- 博客拥有原创内容
- 主页被搜索引擎收录

### 本站信息

```yaml
- name: Petalzu
  link: https://petalzu.top
  description: Face the fear, create the future.
  avatar: https://petalzu.top/images/106128925.webp
```
